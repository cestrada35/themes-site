What inspired this particular theme?

This simple theme was inspired in part by the color schemes of the windows 98 browser. This theme would have made itself right at home at the time, unfortunately I wasn't practicing web developement yet.

Tech used for this project:

  • HTML
  • CSS
  • JavaScript
  • DOM/Fetch/iFrame APIs

I purposely chose to stick with vanilla JavaScript for this project, along with HTML and CSS, to ensure I have a strong foundation of the basics. We can't have a sturdy castle without strong foundations afterall!
